# JNTUH Exam Prep (ExamAid Pro)

ExamAid Pro is a specialized application designed to help JNTUH students prepare for their exams efficiently. It leverages AI to analyze syllabi, provide explanations, and solve exam-related queries.

## Features

- **Syllabus Analysis**: Upload your syllabus image to get a breakdown of topics and study plans.
- **AI Explanations**: Get detailed explanations for exam topics.
- **Chat Solver**: Interact with an AI assistant to solve specific exam questions.
- **Exam-Focused**: tailored for JNTUH curriculum and exam patterns.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd examaid-pro
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- **Framework**: React with Vite
- **Language**: TypeScript
- **UI Components**: shadcn-ui
- **Styling**: Tailwind CSS
- **Backend/Edge Functions**: Supabase Edge Functions (Deno)

## Deployment

This project can be deployed on Vercel or any other static hosting platform. The Edge Functions are hosted on Supabase.
